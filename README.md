## Media Extension

Arikaim CMS media site extension.


#### Requirements   
  * [Arikaim CMS](https://github.com/arikaim/arikaim)
  
  
#### Features
  * Control Panel 
    * Sort media file by status, title, date, plays 
    * Categories
    * Tags
    * Edit media
    * Upload media file 
    * Search media file
    * Disable media
    * Delete media
